# DMLegoArm

SAP Digital Manufacturing Lego Arm Demo

## Pod Plugin

The `pod-plugin/` directory hosts a minimal example client plugin. It includes `manifest.json`, `index.html`, `script.js`, `styles.css`, and `plugin.properties`. Copy these files to your server's plugin directory to enable the plugin. See [`pod-plugin/README.md`](pod-plugin/README.md) for detailed instructions.
